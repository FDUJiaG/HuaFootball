## 首页功能

- **OPR 分数排序** — 点击"分数排序"按钮切换升降序（默认由高到低）
- **评级筛选** — 按 A / B / C / D 四大档快速过滤（A+、A、A- 均归为 A 档）
- **搜索球队** — 支持球队名和评级关键词搜索
- **统计概览** — 自动显示球队总数、平均分、最高/最低分
- **排名编号** — 前三名分别显示 🥇🥈🥉 金色/银色/铜色排名标识

## 添加新球队

只需两步：

1. 将报告 HTML 文件放入 `WorldCup2026/` 目录（文件名格式：`{球队名}-2026-world-cup-report.html`）
2. 在 `config.json` 的 `teamFlags` 和 `teamColors` 中添加该球队的国旗和颜色
3. 运行构建脚本：

```bash
node football-build.js
```

脚本会自动：
- 扫描 `WorldCup2026/` 目录
- 提取每份报告中的 **OPR 评分**（支持 5 种不同报告格式）
- 提取 **评级**（支持 6 种评级写法）
- 生成完整的 `index.html`
- 为每份报告注入「返回首页」按钮

## 国家队评估报告

每份报告基于 **七维度评分体系**（总分 100）：

| 维度 | 权重 | 说明 |
|------|:----:|------|
| ① 阵容实力 | 30% | 球星质量、位置均衡度、板凳深度、身价年龄 |
| ② 战术体系与教练 | 20% | 主帅履历、阵型适配、攻守转换、临场指挥 |
| ③ 近期竞技状态 | 15% | 近12月战绩、出线含金量、俱乐部状态 |
| ④ 大赛经验与心理素质 | 10% | 历史战绩、核心经验值、逆境应对 |
| ⑤ 分组及赛程影响 | 10% | 对手分析、晋级路线、旅途气候 |
| ⑥ 团队凝聚力与场外环境 | 10% | 更衣室团结、足协管理、舆论压力 |
| ⑦ 体能储备与伤病控制 | 5% | 赛季疲劳度、关键伤病、医疗保障 |

## OPR 评分分级

| 分数区间 | 评级 | 含义 |
|:--------:|:----:|------|
| 90+ | A+ | 冠军热门，几乎没有短板 |
| 80-89 | A | 一流强队，四强有力竞争者 |
| 70-79 | B | 二线劲旅，八强实力 |
| 60-69 | C | 具备出线实力 |
| <60 | D | 搅局者 / 鱼腩 |
